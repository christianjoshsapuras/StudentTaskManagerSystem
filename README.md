# Student Task Manager System

## Overview

Student Task Manager System is a simple Java console-based program that helps users manage tasks. The system allows the user to create, view, edit, mark as complete, and delete tasks. Each task includes a title, description, due date, status, and category.

## Features

- Create a task
- View all tasks
- Edit a task
- Mark a task as complete
- Delete a task
- Add a due date and category to a task

## Technologies Used

- Java
- Eclipse IDE

## Classes

- `Main` - handles the menu and user input.
- `Task` - stores task details such as title, description, due date, status, and category.
- `Category` - stores the category name of a task.
- `TaskManager` - manages task operations such as create, view, edit, complete, and delete.
